# ``MaritimeCoordinate``

## Overview

A representation of a geographic coordinate tailored for maritime navigation.

``MaritimeCoordinate`` extends standard coordinate types to include maritime-specific metadata, such as depth and navigational restrictions.

```swift
let coordinate = MaritimeCoordinate(latitude: 37.8080, longitude: -122.4276, depth: 15.5)
```
